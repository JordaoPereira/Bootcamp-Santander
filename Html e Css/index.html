Neste parágrafo eu aprendi que "href" + "target="_blank"" faz o link abrir em uma nova aba, e se eu usar ele sem essa linha de expressão obviamente abrirá na mesma aba.

Ex:
Abrir nova aba
Na mesma aba

Agora para enviar email se utiliza um prefixo após o href "mailto:seuemail"

Ex:
Enviar e-mail

Agora para enviar email se utiliza um prefixo após o href "mailto:seunumero"

Ex:
Ligar agora
